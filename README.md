# CLCL-tool_python
## Python plugin for CLCL clipboard manager
One of the main advantages of CLCL over other other clipboard managers ist the feature to include plugins to extend CLCL's functions.  
Unfortunately the plugin architecture is based on C, and C is not easy to learn. Python on the other hand is an easy-to-learn programming language.  
This plugin intends to enable users of CLCL to create their own plugin.
